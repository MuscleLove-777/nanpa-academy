# 💬 ナンパアカデミー

**声かけから始まるコミュニケーション戦略を体系的に学ぶ学習プラットフォーム**

「ナンパ」を"初対面の相手と信頼関係を最速で構築する技術"として再定義し、マインドセットから実践テクニック、ビジネス転用まで体系的に学べるアカデミーです。

## 📚 カリキュラム

| Level | タイトル | 内容 |
|:-----:|---------|------|
| 🧠 1 | マインドセットと準備 | 内面の準備と外見の磨き方 |
| 🗣️ 2 | 声かけと会話の技術 | 声かけの3大原則、オープナーの型、会話テクニック |
| 📱 3 | 連絡先交換と倫理 | 連絡先交換のタイミング・フレーズと守るべきルール |
| 📲 4 | メッセージと場所別攻略 | LINE攻略、デートの誘い方、場所別アプローチ |
| 🎯 5 | ビジネス転用と成長ロードマップ | ナンパスキルのビジネス転用と上達プラン |
| 🏆 6 | 上達ロードマップと総まとめ | 段階的な成長プランとナンパの本質 |

## 🚀 使い方

1. `index.html` をブラウザで開く
2. ダッシュボードからレベルを選択
3. モジュールを順番に学習
4. 各モジュール末の理解度チェックで知識を確認

## 🛠️ 技術構成

- **HTML / CSS / JavaScript** のみで構成（フレームワーク不要）
- オフラインで動作可能
- ダークモード対応
- レスポンシブデザイン（スマホ対応）
- 学習進捗は `localStorage` に自動保存
